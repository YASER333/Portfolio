
# Mohammed Yaser - Portfolio

A high-end, animated portfolio website built with React, Vite, Tailwind CSS, Three.js, and Framer Motion.

## Features

- **Dynamic Theme System**: Seamless toggle between "Luxury Light" and "Futuristic Dark" modes.
- **Advanced Animations**: Powered by `framer-motion` for smooth scrolling, text reveals, and staggered entrances.
- **3D Interactive Robot**: A fully responsive, procedurally generated 3D robot character that tracks the user's journey through the site, changing poses based on the active section.
- **Glassmorphism UI**: Modern, frosted glass effects on cards and navigation.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Interactive Elements**: Magnetic hover effects, animated skills grid, and project overlays.

## Setup & Run

1.  Install dependencies:
    ```bash
    npm install
    # Note: If three/fiber is missing, run: npm install three @react-three/fiber @react-three/drei
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open `http://localhost:5174` (or port specified in terminal).

## Key Technologies

-   **React**: UI Library
-   **Vite**: Build Tool
-   **Tailwind CSS**: Styling
-   **Framer Motion**: 2D Animations
-   **Three.js / React Three Fiber**: 3D Rendering & Animations
